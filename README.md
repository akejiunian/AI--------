# AI设计质量提升平台

> 面向市政设计院的智能化设计质量控制集成平台，融合知识管理与AI智能工具，贯穿设计全流程，提升设计质量与效率。

## 项目简介

AI设计质量提升平台是一款专为市政设计院打造的综合性智能设计质量管控平台。平台以「三大知识库 + 五大智能工具」为核心架构，结合大语言模型（LLM）、检索增强生成（RAG）、OCR识别等AI技术，为设计人员提供从知识检索、方案设计到图纸审查的全链路智能化支持。

## 核心功能

### 三大知识库

| 模块 | 说明 |
|------|------|
| **技术库** | 国家/行业/地方/企业标准管理、标准图集、材料数据库、智能问答检索 |
| **校审库** | 校审要点、错题案例库、质量红线、知识图谱关联 |
| **项目库** | 历史项目信息、交付物管理、经验库、相似项目智能推荐 |

### 五大智能工具

| 模块 | 说明 |
|------|------|
| **智能标书中心** | AI自动生成标书、模板管理、合规性检查、废标风险检测 |
| **智能图审工场** | PDF批处理、OCR识别、标准符合性检查、图签完整性验证 |
| **智能文审中心** | 合同审查、一致性检查、文本质量检测、合规性验证 |
| **CAD智能设计助手** | 统一插件框架、自然语言交互、专业规则校验、出图标准管理 |
| **智能方案设计引擎** | 三层架构（数据库+规则引擎+LLM生成），覆盖桥梁、道路、污水等多专业 |

## 技术栈

### 前端

- **框架：** Vue.js 3.4
- **路由：** Vue Router 4.3（Hash 模式）
- **架构：** SPA 单页应用，模块化组件设计
- **样式：** 自定义 CSS + CSS Variables 主题系统
- **特点：** 无需构建工具，CDN 加载，开箱即用

### 后端（规划中）

- **LLM：** 本地部署开源大模型（Qwen / GLM）
- **RAG：** 向量数据库 + 语义检索
- **OCR：** 本地部署（PaddleOCR 等）
- **CAD集成：** AutoCAD .NET API / ObjectARX
- **文档处理：** PDFium
- **部署方式：** 私有云 / 本地服务器

## 快速开始

### 环境要求

- 现代浏览器（Chrome / Firefox / Edge）
- 本地 Web 服务器（推荐）

### 启动方式

**方式一：本地服务器启动（推荐）**

```bash
# 进入前端目录
cd frontend

# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000

# 或使用 PHP
php -S localhost:8000
```

然后在浏览器中打开 `http://localhost:8000`

**方式二：直接打开**

直接在浏览器中打开 `frontend/index.html` 即可（部分功能可能受浏览器安全策略限制）。
